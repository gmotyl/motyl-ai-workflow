# Key Decisions

## Project Structure

**Decision:** Use consistent folder structure across all projects

**Rationale:**
- Easier to switch between projects
- Clear where notes, progress, and config belong
- Portable across different AI providers

## Provider Selection

**Decision:** Choose provider based on project context (personal vs. work) and budget

**Rationale:**
- Personal projects: Use free/affordable options
- Work projects: Use tools company sponsors
- Same workflow structure, different tools

## Session Tracking

**Decision:** Track each session in `progress/YYYY-MM-DD-slug.md`

**Rationale:**
- Clear continuity when resuming projects
- Easy to prepare standups
- Searchable history of work
