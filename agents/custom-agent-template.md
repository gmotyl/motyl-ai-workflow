## Configuration

### Global Config

Where to store global credentials:
```bash
export YOUR_PROVIDER_API_KEY='key'
```

### Project-Specific Config

Add to `.agent/config.json`:

```json
{
  "provider": "your-provider",
  "provider_options": {
    "model": "your-model",
    "api_key": "your-key"
  },
  "features": ["feature1", "feature2"],
  "notifications": {
    "enabled": true,
    "style": "system-sounds"
  }
}
```
