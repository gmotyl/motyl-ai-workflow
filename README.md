# motyl-ai-workflow — moved to Pavilio

This project has been renamed and relocated. All active development now happens at:

## → [github.com/gmotyl/pavilio](https://github.com/gmotyl/pavilio)

**Why the rename?**

`motyl-ai-workflow` grew past personal-tooling scope and needed a standalone, globally
pronounceable brand so it can exist as its own product. The new name is **Pavilio** — a coined word
evoking *papilio* (Latin: butterfly — a nod to the Polish "motyl" origin) and *pavilion* (structure,
workshop).

**New homes**

| Resource | Location |
|----------|----------|
| Code | https://github.com/gmotyl/pavilio |
| Website | https://pavilio.ai |
| Docs | https://pavilio.motyl.dev |

**If you had this repo cloned**

```bash
git remote set-url origin git@github.com:gmotyl/pavilio.git
git fetch origin
```

The code and commit history are preserved at the new location. This repo remains online as a
gravestone redirect — all active work happens at [gmotyl/pavilio](https://github.com/gmotyl/pavilio).
